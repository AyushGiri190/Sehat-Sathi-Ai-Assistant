Sehat Sathi is an AI-powered assistant designed to democratize access to **medical exposure and preliminary diagnosis**, thereby addressing the significant disparity in healthcare access across different socioeconomic classes.

## ‚ú® Key Features

* **AI-Powered Consultation:** Utilizes the Google **Gemini 2.0 Flash** Large Language Model for personalized, context-aware, and informative responses to user medical queries.
* **Visual Disease Diagnosis:** Integrates advanced Deep Learning and Transfer Learning models to detect and diagnose diseases from medical images.
* **Full-Stack Application:** Built with a modern, responsive frontend and a robust, scalable backend structure.
* **Diverse Diagnostic Capabilities:** Includes visual diagnosis for:
    * **Brain Tumours**
    * **Breast Cancer**
    * **Lung Cancer**
    * **Tuberculosis (TB)**
